Instalace XXX
=============

It's really easy to start using RTD for your project's documentation. This
section shows you how.

If you are already using Sphinx_ for your docs, skip ahead to
:ref:`import-docs`.

.. image:: images/tshirt.jpg

---------------

    $ cd /path/to/project
